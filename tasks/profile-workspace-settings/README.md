# Feature Pack: Profile, Workspace, and Account Settings

## Goal
Guide backend and frontend agents to implement a complete profile/settings area where users can view workspace context, role, and manage account security settings.

## Assignment Model
- Frontend agent gets:
  - `shared-contracts.md`
  - `frontend/README.md`
- Backend agent gets:
  - `shared-contracts.md`
  - `backend/README.md`

## Must-Have Outcomes
1. User can view profile details.
2. User can see current workspace and current role.
3. User can update profile info.
4. User can request password reset via email (for password-enabled accounts).
5. User can manage core account security settings.
